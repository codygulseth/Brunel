# Product Vision

AI Project Engineer is an AI-assisted construction project-management tool optimized for data-center construction while keeping its core model useful for general commercial projects. The initial product is a deterministic registry that helps humans find the recorded owner of a decision, document, question, or action.

## Product rules

- Every factual output must eventually be traceable to project information.
- Never fabricate drawing requirements, specification requirements, dates, approvals, or responsibilities.
- Clearly distinguish recorded facts, inferred relationships, and suggested actions.
- Never claim to approve work or replace a licensed design professional.
- Require human review before sending external communications or assigning contractual responsibility.
- Preserve source, author, version, and date metadata when project documents are added in later features.
- Prefer simple, deterministic workflows before autonomous agents.
- Optimize for data-center construction while keeping the core model useful for general commercial construction.

The registry records training data; it does not interpret contracts or assign real contractual authority.
